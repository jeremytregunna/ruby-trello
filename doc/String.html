json_into(kls) click to toggle source

Decodes some JSON text in the receiver, and marshals it into a class specified in kls.

For instance: 

  class Stuff
    attr_reader :a, :b
    def initialize(values)
      @a = values['a']
      @b = values['b']
    end
  end
  thing = '{"a":42,"b":"foo"}'.json_into(Stuff)
  thing.a == 42
  thing.b == "foo"
